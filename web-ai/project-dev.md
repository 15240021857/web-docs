# 人工智能代码编辑器 开发项目
## 是什么？
- 传统代码编辑器：前端人员开发代码的编辑器，如vscode,webstorm等
- 人工智能代码编辑器：基于人工智能体的代码生成能力，实现代码的自动补全、代码生成、代码优化等功能
  - 如Cursor
  - 如Trae

## 为什么需要人工智能代码编辑器？
- 根据原型UI，需求，接口文档，快速生成项目代码，大大提高开发效率。
- 让开发人员专注需求，实现交给AI大模型，代码由编辑器自动生成
- 快速排查并修复报错

## 怎么做？
- 基于项目和需求，让AI进行技术选型
- 基于技术选型，让AI创建项目
- 把页面原型图或者UI图，交给AI，写入描述，让AI生成 ** 页面布局 ** 代码
- 业务页面怎么开发？
  - 把用户管理菜单页的UI图交给AI，写入描述，把需求+接口地址，方法，请求参数，响应参数，给到智能体，让他直接生成页面代码
- 出现报错怎么办？
  - 复制或截图报错，交给AI，让它解决即可
