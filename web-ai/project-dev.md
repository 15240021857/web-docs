# 人工智能代码编辑器开发
## 是什么？
- 前端人员开发代码的编辑器，如vscode,webstorm等
- 人工智能编辑器：基于人工智能体的代码生成能力，实现代码的自动补全、代码生成、代码优化等功能
  - 如Cursor
  - 如Trae

## 为什么需要人工智能代码编辑器？
- 提高开发效率：自动补全、代码生成、代码优化等功能，可以减少开发人员的工作量，提高开发效率
- 快速排查并修复报错

## 怎么做？
- 基于项目和需求，进行技术选型
- 基于技术选型，让编辑器的智能体创建项目
- 把页面原型图或者UI图，交给智能体，写入描述，让智能体生成 ** 页面布局 ** 代码
- 业务页面怎么开发？
  - 把用户管理菜单页的UI图交给智能体，写入描述，把需求+接口地址，方法，请求参数，响应参数，给到智能体，让他直接生成页面代码
- 出现报错怎么办？
  - 复制或截图报错，交给智能体，让它解决即可
